# Tokens Optimizer

This skill helps optimize tokens usage in OpenClaw conversations by analyzing conversation patterns and providing practical recommendations.

## Core Features

### 1. Conversation Analysis
- Identify redundant or irrelevant information
- Highlight opportunities for clarification
- Detect unnecessary context repetition

### 2. Optimization Recommendations
- Suggest more concise phrasing
- Recommend splitting complex questions
- Provide context management strategies

### 3. Usage Tracking
- Monitor tokens consumption trends
- Record optimization results
- Provide usage statistics

### 4. Tokens Estimation
- Calculate estimated tokens usage from conversation text
- Provide cost estimation based on current pricing
- Display detailed usage information

## How to Use

### 1. Automatic Analysis
- The skill automatically analyzes conversations
- Highlights optimization opportunities in real-time
- Provides suggestions for improvement

### 2. Manual Optimization
- Ask for specific optimization recommendations
- Request conversation analysis
- Get tips on tokens-efficient communication
