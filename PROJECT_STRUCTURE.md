## 🎯 Directory Purpose

### `/skills/`
Each subdirectory is a complete OpenClaw skill with:
- `SKILL.md` - Skill definition and metadata
- `README.md` - Detailed documentation and marketing
- `scripts/` - Python scripts and implementations
- `references/` - Additional documentation and strategies

### `/docs/`
Additional documentation for users:
- Installation guides
- Usage examples
- Cost savings calculator
- Best practices

### `/archive/`
Old workspace files kept for reference only:
- Not part of the main skills
- Can be deleted if not needed
- Preserved for historical context

## 📝 Skill Structure

Each skill follows this standard structure:

```
skill-name/
├── SKILL.md              # OpenClaw skill metadata
├── README.md             # User documentation
├── scripts/              # Implementation code
│   └── main_script.py
└── references/           # Additional docs
    ├── strategies.md
    └── examples.md
```

## 🚀 How to Use

1. Clone or download this repository
2. Copy the `skills/` folder to your OpenClaw workspace
3. Restart OpenClaw or reload skills
4. Skills activate automatically!
