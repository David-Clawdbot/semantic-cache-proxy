# 🚀 Tokens Optimization Skills Collection

Save 50-70% on your token costs with these powerful OpenClaw skills!

## 💡 Overview

This collection contains 6 powerful skills designed to optimize token usage and reduce costs while maintaining conversation quality.

## 🎯 Skills Included

### 1. 🔧 tokens-optimizer
**Analyzes and optimizes conversation patterns for better token efficiency.**

- **Features**: Conversation analysis, optimization recommendations, usage tracking
- **Savings**: Foundation for all optimization
- **Keywords**: "save tokens", "optimize conversation", "tokens usage"
- **Trigger**: Automatic on relevant keywords

---

### 2. 🧠 semantic-cache-proxy
**Semantic caching for repeated queries - similar questions return cached results.**

- **Features**: Semantic similarity matching (≥0.85), Redis storage, 24-hour expiration
- **Savings**: Up to 80% on repeated queries
- **Keywords**: "cache", "semantic search", "repeat questions"
- **Trigger**: Automatic

---

### 3. 📝 conversation-compression
**Compress long chat history into key summaries using /compact command.**

- **Features**: Automatic compression at 8+ rounds, key point extraction, decision tracking
- **Savings**: 70-90% on history tokens
- **Keywords**: "save tokens", "shorten conversation", "/compact"
- **Trigger**: Auto at 8+ rounds or manual /compact

---

### 4. 🔄 model-auto-switching
**Automatically switch to low-cost models for simple tasks, high-cost for complex ones.**

- **Features**: Smart task analysis, multi-model support, cost estimation
- **Savings**: ~30% overall (up to 75% on simple tasks)
- **Keywords**: "model switch", "cost optimization", "simple task"
- **Trigger**: Automatic per task

---

### 5. ✂️ system-prompt-slimming
**Slim down redundant rules in configuration files, keep under 2000 words.**

- **Features**: Redundancy detection, content merging, size enforcement
- **Savings**: 65% on system prompt tokens
- **Keywords**: "slim prompt", "optimize config", "reduce system"
- **Trigger**: Automatic on startup or /slim-prompt

---

### 6. 🪟 context-window-manager
**Smart context window management with sliding windows and intelligent summarization.**

- **Features**: Sliding window (6-8 messages), importance marking, auto-summary
- **Savings**: 50-80% on long conversations
- **Keywords**: "window", "context", "/keep", "/window"
- **Trigger**: Automatic at 8+ rounds

---

## 🚀 Installation

1. Clone or copy the skills to your OpenClaw workspace
2. Skills are auto-detected in `~/.openclaw/workspace/skills/`
3. Restart OpenClaw or reload skills

---

## 💡 Usage

Skills activate automatically based on conversation context. Manual commands available:

- `/compact` - Compress current conversation
- `/window [n]` - Set context window size
- `/keep` - Mark current message as important
- `/slim-prompt` - Optimize system prompts
- `/cost-estimate` - Show cost savings
