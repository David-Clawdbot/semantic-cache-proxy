# Tokens 使用优化建议

## 🎯 核心原则
保持对话 **清晰、简洁、有针对性** 是节省 tokens 的最佳方法。

## 📝 优化策略

### 1. 提问/指令优化
- **直接明确**：避免模糊描述，直接说需求
- **分段处理**：复杂任务分步骤解决
- **避免重复**：已提供的信息不必反复说明

### 2. 对话格式建议
**✅ 推荐：**
> "帮我写一个 Python 函数，计算列表的平均值"

**❌ 避免：**
> "我需要一个函数，那个，就是能算平均值的，用 Python 写，因为我要处理列表数据，你知道吧..."

### 3. 系统配置优化
- 保持 OpenClaw 运行在最新版本
- 定期清理内存文件中不需要的历史记录

## 📊 消耗监控
每次会话结束后，我会自动检查内存占用情况，并记录到每日笔记中。
